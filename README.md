# Ezags::Xsd

mappers for ezags-protocols

## Installation

Add this line to your application's Gemfile:

    gem 'ezags-xsd'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ezags-xsd

## Usage

    $ ezags-xsd help
